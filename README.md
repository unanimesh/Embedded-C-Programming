# Embedded C Programming - STM32 Learning Journey

## Overview

This repository contains my hands-on work, exercises, projects, and notes while learning **Embedded C Programming** through the Udemy course by **Kiran Nayak**.

The goal of this repository is to document my progress in embedded systems development, firmware programming, and STM32 microcontroller programming using industry-standard tools and workflows.

## Hardware Platform

* STM32 Nucleo-F411RE Development Board
* ARM Cortex-M4 Microcontroller
* On-board ST-LINK Debugger/Programmer

## Development Environment

### Operating System

* Debian Trixie (XFCE Desktop)

### IDE & Tools

* STM32CubeIDE 2.1.1
* GCC ARM Toolchain
* STM32CubeMX
* Git & GitHub


## Topics Covered

### Embedded C Fundamentals

* Variables and Data Types
* Storage Classes
* Pointers
* Structures and Unions
* Bit Manipulation
* Preprocessor Directives
* Memory Layout
* Static and Dynamic Memory Concepts

### ARM Cortex-M Programming

* Processor Architecture
* Registers
* Stack Operations
* Exception Handling
* Interrupts
* NVIC

### STM32 Peripheral Programming

* GPIO
* UART
* SPI
* I2C
* Timers
* ADC
* DMA
* PWM
* RTC
* External Interrupts

### Debugging

* ST-LINK Debugging
* Breakpoints
* Watch Variables
* Register Inspection
* Memory Analysis

## Learning Objectives

* Develop strong Embedded C programming skills.
* Understand ARM Cortex-M architecture.
* Learn bare-metal firmware development.
* Interface and control STM32 peripherals.
* Build real-world embedded applications.
* Follow professional embedded software development practices.
